# StudyNotion - An Ed-Tech Platform

**Live Demo:** [StudyNotion](https://study-notion-frontend-gamma-red.vercel.app)

## 📖 Project Description
StudyNotion is a fully functional ed-tech platform that enables users to create, consume, and rate educational content. The platform aims to provide a seamless and interactive learning experience for students, making education more accessible and engaging, while giving instructors a platform to showcase their expertise.

## 🚀 Tech Stack
The platform is built using the MERN stack:
* **Front-end:** ReactJS, Tailwind CSS, Redux for state management.
* **Back-end:** NodeJS and ExpressJS utilizing a monolithic architecture.
* **Database:** MongoDB.

## ✨ Key Features
* **User Authentication:** Secure signup and login for students and instructors, complete with OTP verification and password reset capabilities.
* **Course Management:** Instructors can create, read, update, and delete courses, while students can view and rate them.
* **Payment Integration:** Seamless checkout flow integrated with Razorpay for course enrollment.
* **Cloud Media Management:** Integration with Cloudinary to store and manage all media content, including images and videos.
* **Markdown Formatting:** Course content is stored and rendered using Markdown for easier display.

## 🌍 Deployment Infrastructure
* **Front-end:** Vercel.
* **Back-end:** Render.
* **Database:** MongoDB Atlas.
* **Media Storage:** Cloudinary.

## 🔮 Future Enhancements
* **Gamification:** Adding badges, points, and leaderboards to increase user engagement.
* **Personalized Learning Paths:** Creating tailored paths based on student interests.
* **Mobile App:** Expanding reach and convenience with a dedicated mobile application.
